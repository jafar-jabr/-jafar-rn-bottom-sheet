# @jafar/rn-bottom-sheet

a bottom sheet for react native apps

## Installation

```sh
npm install @jafar/rn-bottom-sheet
```

## Usage

```js
import RnBottomSheet from "@jafar/rn-bottom-sheet";

// ...

const result = await RnBottomSheet.multiply(3, 7);
```
